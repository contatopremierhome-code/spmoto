## Style Guidelines:

- Background color: White (#FFFFFF) to provide a clean and clear visual field.
- Primary color: A vibrant green (#34A853), symbolizing action, approval, and directness, in harmony with a frequent UI color for Whatsapp.
- Accent color: A bright lime green (#AEEA00), used to create visual interest and emphasis, with good contrast for primary buttons.
- Font: 'Inter' (sans-serif) for both headlines and body text, selected for its neutral, modern aesthetic, suitable for mobile legibility and clear messaging.
- Employ green icons to highlight key features and solutions.
- Mobile-first design, prioritizing vertical screen optimization.
- Subtle transitions on the image carousel and quantity selector for enhanced user interaction.
